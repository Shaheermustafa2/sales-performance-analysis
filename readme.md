### **Sales Performance Analysis Dashboard**



#### **Project Overview**

* This project analyzes sales, profit, and regional performance using PostgreSQL and Power BI.



#### **Tools**

* PostgreSQL
* SQL
* Power BI
* Excel



#### **Dashboard Pages**

* Sales Overview
* Product \& Category Analysis
* Regional Performance



#### **Key Insights**

* Central region contributes highest revenue.
* Some sub-categories generate losses.
* Monthly sales show declining trend.
